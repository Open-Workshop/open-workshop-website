Что такое Open Workshop?

Это open-source проект с открытым API и кодом.

Для чего он нужен?

Скачивайте, обновляйте и ищите моды без ограничений!

Какие проекты можно реализовать при помощи OW?

  • Сайт - обычный web сайт для поиска и скачиваний модов, например как этот 🤓
  • Боты - вы можете интегрировать каталог сервиса например в Discord, Telegram или другую соц. сеть!
  • Настольный клиент - сервис позволяет как скачивать, так и проверять обновления для модов - т.е. можно создать лончер!
    •  Или можете придумать свою идею и реализовать её!
